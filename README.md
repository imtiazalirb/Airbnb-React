## Run Locally

Clone the project

```bash
  git clone /https://github.com/imtiazalirb/Airbnb-React.git
```

Go to the project directory

```bash
  cd Airbnb-React
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
